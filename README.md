# gsheets

Google Sheets skill focused on formulas, Apps Script, debugging, and Sheets-native analytical workflows.

## What this repo covers

- Google Sheets formulas and formula debugging
- Apps Script automation
- Google Sheets-native financial analysis patterns
- Converting spreadsheet logic into Sheets syntax
- Dashboard and model layout guidance for Google Sheets
- Test prompts and expected behaviors for evaluation

## Repo layout

- `SKILL.md`: main trigger and operating instructions
- `examples/`: worked examples and response patterns
- `resources/`: cheatsheets and practical guidance
- `tests/`: prompts and review criteria

## Positioning

This skill should trigger when Google Sheets is the target platform.
It should not replace an Excel/xlsx workflow when the required deliverable is an `.xlsx` file.

## What makes this stronger

- Clear response modes
- Concrete formula patterns
- Apps Script delivery conventions
- Finance-oriented Sheets examples
- Quality checks and anti-patterns
- Test prompts for evaluation
