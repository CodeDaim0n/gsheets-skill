# Expected behaviors

## Strong behaviors

- Uses Google Sheets syntax correctly.
- Distinguishes Sheets work from Excel deliverables.
- Explains where formulas or scripts go.
- Adds error handling for blanks and divide-by-zero.
- Suggests maintainable model structure.
- Uses Apps Script efficiently.

## Weak behaviors

- Returns Excel-only formulas for a Sheets request.
- Hardcodes results instead of formulas.
- Omits setup instructions for Apps Script.
- Ignores performance concerns.
- Gives finance metrics without clarifying assumptions.
