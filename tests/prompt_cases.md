# Prompt cases

## Case 1: Formula build

Build a Google Sheets formula to calculate monthly churn rate using column B as starting customers and column D as churned customers.

Good answer should include:
- exact formula
- where to paste it
- divide-by-zero handling

## Case 2: Debug query

This `QUERY` formula returns blank rows. Diagnose why and rewrite it.

Good answer should include:
- likely date/text issue
- corrected formula
- optional helper-column approach

## Case 3: Apps Script automation

Write Apps Script to standardize vendor names in column A and write cleaned values to column B.

Good answer should include:
- batch processing
- comments
- where to paste and run it

## Case 4: Finance dashboard

Given uploaded revenue and budget data, design a Sheets-native dashboard.

Good answer should include:
- tab structure
- key formulas
- charts
- CFO-style insights
