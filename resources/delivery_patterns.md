# Delivery patterns

## Formula response pattern

1. State the destination cell.
2. Give the formula.
3. Explain assumptions about source columns.
4. Note any one-time action such as approving `IMPORTRANGE`.
5. Offer an error-safe alternative if needed.

## Apps Script response pattern

1. Give the full script.
2. Explain where to paste it.
3. Explain how to run it.
4. Mention permissions and testing advice.
5. Mention trigger setup if needed.

## Dashboard response pattern

1. Recommend tab structure.
2. List KPIs.
3. Suggest chart types.
4. Separate raw data from calculations.
5. Keep formulas maintainable.
